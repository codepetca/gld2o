# GLD2O Project Docs

Purpose: GLD2O course package and link-only Next.js site for students accessing assignments via Google Classroom. This folder summarizes conventions, structure, and update steps for future edits or AI contributors.

Contents:
- `ai_playbook.md` — guidelines for AI assistance, editing conventions, and release flow.
- `content_model.md` — how assignments/resources are stored and rendered on the site.
- `workflow.md` — Google Classroom + Pika + Vercel deploy steps.
